CSV Format Instructions

Your CSV file should have the following columns:

Column Required Description
first_name Required Student's first name
last_name Required Student's last name
class_name Optional Class (e.g., "Basic 7", "Basic 8")
age Optional Age in years (default: 10)
Example CSV:
first_name,last_name,class_name,age John,Doe,Basic 7,12 Jane,Smith,Basic 8,13 Michael,Johnson,Basic 7,12 Sarah,Williams,,11
Note: If class_name is not provided or doesn't exist, students will be assigned to the default class you select above.
