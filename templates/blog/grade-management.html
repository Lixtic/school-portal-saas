How SchoolPadi Automates
Grade Management

Enter class scores and exam scores. The system computes weighted totals, assigns grades, writes remarks, and renders polished PDF report cards — individual or full class — in one click.

4 min read SchoolPadi Team

The problem with manual grading

Every term, teachers across Ghana spend countless hours computing totals, converting raw scores to grades, and handwriting report cards. It's repetitive, error-prone, and takes time away from actual teaching. A single miscalculation can affect a student's entire term outcome.

SchoolPadi eliminates this entirely. Teachers enter two numbers per student per subject — class score and exam score — and the system handles everything else.

How automatic grade calculation works

When a teacher enters scores into SchoolPadi, the following happens automatically:

  • Weighted total — Class score and exam score are combined using the school's configured weighting (e.g., 30/70 or 40/60)
  • Grade assignment — The total is mapped to the appropriate GES grade (A1 through F9 for JHS, or the primary scale)
  • Remarks generated — Each grade receives an automatic remark (Excellent, Very Good, Good, etc.)
  • Analytics computed — Class averages, subject performance trends, and rank positions are calculated in real-time

Configurable weighting per subject

Different schools weight class assessments and exams differently. Some use a 30/70 split, others prefer 40/60. SchoolPadi lets administrators configure these percentages per subject, so the calculation always matches the school's policy.

This means a Science class can use 30% class score and 70% exam, while Creative Arts uses 50/50 — all within the same school, same term.

GES grading systems — Primary, JHS, and SHS

SchoolPadi supports the Ghana Education Service grading scales out of the box:

  • Primary — Level-based descriptors for younger learners
  • JHS — A1 through F9 grading scale with 9-point system
  • SHS — Compatible with WAEC/BECE grading requirements

Schools select their grading system during setup. Teachers never need to think about grade boundaries — the system applies them consistently, every time.

One-click PDF report cards

Once grades are entered, SchoolPadi generates professional PDF report cards that include the student's scores, grades, remarks, class position, teacher and headmaster comments, and the school's branding.

Teachers can generate a single student's report card or download the entire class in one click. No more handwriting. No more template formatting. Just data in, report out.

Grade analytics — trends, ranks, and insights

Beyond individual report cards, SchoolPadi provides analytics that help teachers and administrators understand performance patterns:

  • Class averages — How is each class performing overall?
  • Subject trends — Which subjects are improving? Which need attention?
  • Rank lists — Automatic positioning based on cumulative performance
  • Term-over-term comparison — Track student progress across the academic year
