CSV Format Instructions

Your CSV file should have the following columns:

Column Required Description
first_name or First Name Required Teacher's first name
last_name or Last Name Optional Teacher's last name
employee_id or Employee ID Optional Employee ID (auto-generated if blank)
email or Email Optional Email address (auto-generated if blank)
phone or Phone Optional Phone number
qualification or Qualification Optional Academic qualification (default: Bachelor of Education)
age or Age Optional Age in years (default: 30)
date_of_joining or Date of Joining Optional Date joined (YYYY-MM-DD format, default: today)
Example CSV:
First Name,Last Name,Employee ID,Email,Qualification,Age,Date of Joining John,Doe,TCH001,john.doe@school.com,Master of Education,35,2020-01-15 Jane,Smith,,jane.smith@school.com,Bachelor of Science,28,2021-09-01 Michael,Johnson,TCH003,,PhD in Mathematics,42,2019-03-10 Sarah,Williams,,,Bachelor of Education,30,
Flexible Format: The import supports both space-separated column names (like "First Name") and underscore-separated names (like "first_name"). Missing values will be auto-generated where applicable.
Important: Employee IDs must be unique. If an employee ID already exists in the system, that row will be skipped.
